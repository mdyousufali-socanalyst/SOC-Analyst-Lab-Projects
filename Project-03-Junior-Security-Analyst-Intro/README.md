# Junior Security Analyst Intro

This project documents my hands-on experience with foundational SOC tasks on TryHackMe.

## Key Learning Points
* **Alert Analysis:** Investigating malicious IP addresses and identifying potential threats within the SIEM dashboard.
* **Escalation Process:** Understanding the importance of hierarchical responsibility by escalating critical alerts to the SOC Team Lead.
* **Firewall Configuration:** Learning how to effectively block malicious IPs to mitigate active attacks on the network.

## Tools & Skills
* SOC Operations
* Alert Triage
* Firewall Management
* Log Analysis
